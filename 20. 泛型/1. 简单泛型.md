## 持有泛型字段的类的定义-泛型类
* 在类名后，用`<T>`声明类型占位符  
```java
public GenericHolder<T> {
  private T a;
  public GenericHolder() {}  //构造函数后不需要加 类型占位符<T>
  public void set(T a) {
    this.a = a;
  }
  public T get() {
    return this.a;
  }
}
```
## 用泛型实现 堆
```java
public class LinkedStack<T> {
	public class Node<U> {
		private U item;
		private Node<U> next;
		public Node() {
			this.item = null;
			this.next = null;
		}
		public Node(U item, Node<U> next) {
			this.item = item;
			this.next = next;
		}
		
		public boolean end() {
			return item == null && next == null;
		}
	}
	
	public Node<T> top = new Node<T>();
	
	public void push(T item) {
		top = new Node<>(item, top);
	}
	
	public T pop() {
		T result = top.item;
		if(!top.end()) {
			top = top.next;
		}
		return result;
	}
	
	public static void main(String[] args) {
		LinkedStack<String> stack = new LinkedStack<String>();
		for(String ss : "jia shao kun".split(" ")) {
			stack.push(ss);
		}
		while(!stack.top.end()) {
			System.out.println(stack.pop());
		}
	}
}
```
